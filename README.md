# Shopping Behavior Analysis and Sales Trend Forecasting
## Overview
This project, developed as an internship assignment, offers a comprehensive deep dive into e-commerce sales data to analyze shopping behaviors and extract actionable business intelligence. The core objective was to explore sales patterns, monthly revenue trends, and best-selling product categories using Python. It was developed as part of the submission for IDEAS TIH Project.

# Tech Stack
1. Python 3.13.7
2. Pandas
3. NumPy
4. Matplotlib
5. Seaborn
# Features
1. Full-Spectrum Revenue Decomposition
2. Exploratory Data Analysis (EDA) with visualisations
3. Time-series trend analysis
4. Sales distribution insights
